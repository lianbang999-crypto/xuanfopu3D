## 来源与方法（诚实说明）

### 来源一：大安法师讲集（佛悦文库）

- 语料：`public/text/` 全部 **38 部讲记 241 篇 + 问答 820 条**（约 1700 万字符）。
- 方法：段落级关键词评分提取（`scripts/extract-jile-scenes.py`）。词典分两级：
  **S1 高精度场景词**（七宝池、八功德水、迦陵频伽、道场树、疑城…命中即取）与
  **S2 一般场景词**（莲华、楼阁、光明…需两词以上叠加或有极乐/净土语境佐证）。
  连续命中的相邻段落合并为一条摘录。
- 分级：含 S1 词的 **629 条为核心摘录**，已全文收入四卷；其余 **669 条为扩展摘录**（仅索引，
  全文在 `data/daan_excerpts.json`）。
- **局限**：关键词法可能仍有少量漏提（不含任何场景词的意译式描述）与误收
  （如泛论三十二相而非专说极乐者）；扩展索引即供人工复筛。摘录一律逐字复制，未做任何改写。

### 来源二：CBETA 经典（cbeta-org/xml-p5 官方 GitHub 镜像，master 分支）

- **净土五经一论**：阿弥陀经（T0366）、观无量寿佛经（T0365）、无量寿经（T0360）、
  称赞净土佛摄受经（T0367，玄奘异译）、势至圆通章（T0945 卷五节录）、
  普贤行愿品（T0293 卷四十节录）、往生论（T1524）。
- **无量寿经四种异译**：汉译平等觉经（T0361）、吴译大阿弥陀经（T0362）、
  宋译庄严经（T0363）、王日休校辑大阿弥陀经（T0364）——异译中的场景细节常可互补。
- **祖师著述八种**：往生论注（昙鸾 T1819）、略论安乐净土义（昙鸾 T1957）、安乐集（道绰 T1958）、
  观经四帖疏（善导 T1753）、净土十疑论（智者 T1961）、妙宗钞（知礼 T1751）、
  龙舒净土文（王日休 T1970）、弥陀要解（蕅益 T1762）。
- 转换规则（`scripts/cbeta-p5-to-text.py`）：**校勘取底本（lem），脚注不混入正文，缺字按
  CBETA 字表映射（无映射以 □ 占位）**；全部标题已逐部核实（如 T1753 确为「沙門善導集記」）。
- 描述段提取同用上述词典（繁→简仅用于匹配，**引文一律保持繁体原样**）。

### 铁律

- 经典原文**不可篡改**：本库一切经文均逐字来自 CBETA；讲集摘录均逐字来自佛悦文库文本。
- 引用请带出处（经名/卷次/段号，或 系列/讲次/段号），段号均可回查原文文件。

## 重跑 / 扩展

```bash
# 1. 重新提取讲集摘录（词典在脚本顶部 LEXICON，可继续加词调级）
python3 scripts/extract-jile-scenes.py

# 2. CBETA XML → 纯文本（如需新增经目，先下载 XML）
python3 scripts/cbeta-p5-to-text.py <xml文件...> -o <输出目录>

# 3. 重新生成资料库全部 markdown
python3 scripts/build-jile-library.py <cbeta_txt目录>
```

新增经目下载（GitHub 镜像直链）：
`https://raw.githubusercontent.com/cbeta-org/xml-p5/master/T/<册>/<册n编号>.xml`
