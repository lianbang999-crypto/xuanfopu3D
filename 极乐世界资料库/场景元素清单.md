## 一、空间总构（世界层次）

经典给出的空间是**分层**的，与敦煌净土变的构图法一致：

| 层次 | 内容 | 经据 |
|---|---|---|
| 地 | 黄金为地；琉璃宝地上以黄金绳界道 | 《阿弥陀经》「常作天樂，黃金為地」；《观经》地想「琉璃地上，以黃金繩，雜廁間錯，以七寶界，分齊分明」 |
| 界围 | 七重栏楯、七重罗网、七重行树 | 《阿弥陀经》「七重欄楯、七重羅網、七重行樹，皆是四寶周匝圍繞」 |
| 水 | 七宝池、八功德水、金沙池底、四边阶道 | 《阿弥陀经》「極樂國土有七寶池，八功德水充滿其中，池底純以金沙布地。四邊階道，金、銀、琉璃、頗梨合成」 |
| 岸上 | 讲堂、精舍、宫殿、楼观；栴檀树 | 《无量寿经》「講堂、精舍、宮殿、樓觀皆七寶莊嚴，自然化成」 |
| 虚空 | 宝楼悬空、天乐、华雨、飞天诸天 | 《观经》「五百億寶樓，其樓閣中有無量諸天」；《阿弥陀经》「晝夜六時，天雨曼陀羅華」 |
| 中心 | 道场树（菩提树）与佛座 | 《无量寿经》第二十八愿：道场树「高四百萬里」 |

> 敦煌净土变通行构图：中央佛会（阿弥陀佛+二胁侍+圣众）、前部平台乐舞、下方宝池化生、
> 上方虚空楼阁华盖飞天——正好逐层对应上表，可直接作画面分层依据。

## 二、依报元素明细

| 元素 | 经典依据（原文） | 视觉要点 |
|---|---|---|
| 宝树 | 《无量寿经》「七寶諸樹周滿世界」（金树、银树、琉璃树…七宝间错） | 单一材质树与多宝合成树两类；树列成行，行行相值 |
| 树的尺度 | 《观经》树想「一一樹高八千由旬」 | 超人尺度，画面中树冠可入云层 |
| 莲花 | 《阿弥陀经》「池中蓮花，大如車輪，青色青光，黃色黃光，赤色赤光，白色白光，微妙香潔」 | 四色四光，**花自发光**而非反光；车轮为最小尺度 |
| 池水 | 《无量寿经》黄金池底白银沙、白银池底黄金沙…（池沙异色互衬） | 水「湛然盈滿，清淨香潔」，可随人意深浅冷暖 |
| 楼阁 | 《阿弥陀经》阶道上「上有樓閣，亦以金、銀、琉璃、頗梨、車𤦲、赤珠、馬瑙而嚴飾之」 | 七宝严饰；部分楼阁悬于虚空 |
| 化鸟 | 《阿弥陀经》「白鵠、孔雀、鸚鵡、舍利、迦陵頻伽、共命之鳥…晝夜六時出和雅音」 | 迦陵频伽（人首鸟身）敦煌乐舞图中常见；鸟为佛所化，非罪报所生 |
| 德风 | 《无量寿经》「德風徐起微動，其風調和，不寒、不暑；溫涼柔軟」 | 风是恒常的轻微动态：树摇、网动、衣带、水纹 |
| 天乐华雨 | 《阿弥陀经》「常作天樂…晝夜六時，天雨曼陀羅華」 | 空中不鼓自鸣的乐器（敦煌净土变定型母题）+ 缓落花雨 |
| 光明 | 《阿弥陀经》「彼佛光明無量，照十方國無所障礙」；《无量寿经》「無量壽佛威神光明最尊第一」 | 全场景无影调死角；光源是佛身与万物自光，非日月 |
| 衣食自然 | 《无量寿经》「百味飲食自然盈滿。雖有此食實無食者」 | 器皿饮食应念浮现、意足自去（可做一个奇观镜头） |

## 三、光色体系

- 基础材质色：金、银白、琉璃青、颇梨（水晶）透明、赤珠红、玛瑙纹、紫金——即**七宝色谱**。
- 莲花四色四光（青/黄/赤/白，色即是光）。
- 地面：金地 + 琉璃地上金绳界道（青金对比，敦煌壁画中即石青底泥金线）。
- 与敦煌矿物色对应：石青、石绿、朱砂、泥金、白垩——此前研究已有「敦煌矿物色×明度谱系」结论，可沿用。
- 关键原则：**万物自光**。渲染上应以自发光+柔光为主，避免强投影的写实日光感。

## 五、动态场景（分镜种子）

1. **日想起观**（转场入西方）：《观经》「見日欲沒，狀如懸鼓」——落日如悬鼓，是十六观第一观，天然的片头。
2. **临终接引**：《阿弥陀经》「其人臨命終時，阿彌陀佛與諸聖眾，現在其前」；《观经》「無量壽佛住立空中，觀世音、大勢至，是二大士侍立左右」——佛立空中的「来迎」构图，敦煌与日本来迎图皆有定式。
3. **莲华化生**：池中莲苞绽开，化生童子/行者合掌而出（华开见佛）。
4. **微风吹树**：《阿弥陀经》「微風吹動，諸寶行樹及寶羅網出微妙音，譬如百千種樂同時俱作」——风过处树网齐鸣，音画同源的核心镜头。
5. **鸟演法音**：杂色鸟六时和雅音，「其土眾生聞是音已，皆悉念佛、念法、念僧」。
6. **清旦供佛**：《阿弥陀经》「常以清旦，各以衣裓盛眾妙華，供養他方十萬億佛；即以食時，還到本國，飯食經行」——晨光、衣裓盛花、瞬行十万亿土往返的时空奇观。
7. **天雨曼陀罗华**：昼夜六时花雨缓落，金地上花不积秽。
8. **饭食经行**：食毕经行于金地林间。
9. **道场树远景**：树高四百万里的中心圣树，众会围绕。
10. **总揽**：《往生论》「觀彼世界相，勝過三界道」——从高空俯瞰整个世界的收束镜头。

## 六、音声体系

水声、树网声、鸟声、天乐皆是法音：《阿弥陀经》鸟音「演暢五根、五力、七菩提分、八聖道分」。
配乐方向：一切环境声最终归于念佛念法念僧的清净感（可与佛号声部融合）。

## 八、使用原则

1. **经文不可改**：视觉化可以取舍详略，引用文字必须原样。
2. **造像存恭敬**：佛菩萨形象以敦煌白描底稿再创作（2026-07-11 已定的条件），不作戏谑变形。
