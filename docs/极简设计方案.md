# 选佛谱 · 极简设计方案（统一规划）

> 2026-07-20 定稿。约束此后所有 UI/UX、美术改动；改此方案先改本文件。
> 总纲：**星图即棋盘，谱文即内容——界面只做三件事：导航、掷轮、读谱。其余一概退隐。**

## 一 · 设计原则

1. **一屏三区**：左杆（场景导航）· 右杆（十五门天梯）· 底栏（掷轮行动）。中央永远留给星图世界，不放常驻控件。
2. **色不过三**：金（#d7aa45 主）· 纸（#efe0b4 文）· 青（#96e1d6 辅，联机/提示专用）。法界配色只在珠色与色点上出现，不做大面积色块。
3. **光即层次**：层级用透明度与发光强弱表达，不用边框粗细；大面积光斑一律 ≤0.42 透明度。
4. **字级五档**：`--fs-xs/sm/md/lg/xl`（11/12.5/14/16/19px）+ 展示级 `--fs-display`（22px）。全站不得出现散点字号；大字模式整体缩放变量。
5. **字族两用**：得意黑（SmileySans）为全站主字；展示级题字（面板标题/落位大字/门介）回退宋体系，气质不塌。
6. **谱义至上**：任何 UI 文案引用谱文必逐字（`scripts/crosscheck-pu.mjs` 护栏）；界面装饰不得遮谱文。

## 二 · 布局架构（一屏三区）

```
┌──────────────────────────────────┐
│ 题字(左上)                菜单(右上) │
│ 场                             天 │
│ 景  ← 中央星图世界（唯一主角） →   梯 │
│ 导                             十 │
│ 航                             五 │
│ (左杆·场景内才现)              门 │
│                                  │
│      [⋯]  长按掷轮  [问]   (底栏) │
└──────────────────────────────────┘
```

- **右杆天梯**（常驻）：十五门刻度+现居门发光；门号一律横排（nowrap）。
- **左杆场景导航**（入场景才现）：色界＝四禅四签；菩萨道场＝十科签。色点+名，无底无框，选中亮点放大。与右杆同语言。
- **底栏**：左「⋯」谱务 · 中「长按掷轮」 · 右「问」。联机时左下加「聊」浮钮。
- **探底剖面杆**：只在娑婆全图存在；入门/入场景/入净土一律隐。

## 三 · 15门220位 交互规范

- **掷轮**：长按默念→松手旁掷（占察轮 3D 实转）；未轮到（联机）按钮压暗不可掷。
- **判词窗**（每掷必出，中央下方）：`轮面二字 → 方向徽 → 去向位名（展示级字）→ 谱曰缘由（可展开原文）→ 行▸`。点正文展白话+原文；下滑收签、上滑唤回。
- **行棋**：彗星光弧乘行，跨门先拉远同框再进；落位金环涟漪+位名大字（展示级）。
- **位次浏览**：点门星展本门位珠；位珠点开＝谱注抽屉（原文+白话+出处行号）。
- **不打断原则**：一切浮层可点外关闭；行棋动画可点跳过（设置「行棋特效」关＝自动直达）。

## 四 · 十法界世界美术规范（极简）

- **须弥山**：敦煌细腰旋成体，四宝四面（南琉璃/东白银/北黄金/西颇胝迦），金线描边为唯一描边语言。
- **七金山七香水海**：金脊环（皆金所成，双金调相间）×7，环距内山基不叠压，海环必可见。
- **四洲**：东半月/南车箱/西满月/北方座+各二中洲；洲色映本面宝色，昼夜随日。
- **色界四禅**：光盘+星环，盘晕 ≤0.42、侧晕减半；成员星只在该层绽开时亮名。
- **无色界**：无方所——只以轴顶微光与名示意，不做实体建筑。
- **净土**：横轴西方（横超光路），敦煌壁画底幕+莲池双层金碧微光。
- **降噪三禁**：禁常亮大光斑、禁多层重叠辉光、禁与谱文争眼的动画。

## 五 · 弹窗（overlay）内容布局规范

- 统一壳：`.panel`（圆角14 / 边线金 26% / 磨砂）+ 遮罩 66%。
- 结构次序（自上而下恒定）：**题（--fs-xl）→ 副题/出处（--fs-xs 灰金）→ 正文（--fs-md 行高1.85）→ 行动钮（gbtn，主钮 primary）**。
- 谱注类弹窗：原文（繁体逐字，含原文括注）在前，白话在后灰阶；出处标 `B0136_00X:行号`。
- 一屏一窗：新窗必关旧窗；✕ 与蒙层点击等效。

## 六 · 数据护栏（对齐原文）

- 底本：CBETA B0136《選佛譜》六卷（`选佛谱G版/繁体版/`）；人工核对层：`xuanfopu-h5/data/positions.json`。
- 护栏命令：`node scripts/crosscheck-pu.mjs`——220 位去向/起手/门题/谱曰须全零差异方可发版。
- 2026-07-20 已修正三处历史错误（均经原文逐字裁定）：
  1. 上品上生「佛佛」→ 極果位（原误：常寂光淨土位）——原文「佛佛。寂光上上」

  另：终局位显示名对齐原文首行「佛」（圓教究竟妙覺位为副题；id 不变，存档兼容）；220 位谱曰全量回填人工核对底本（保留原文夹注与科判结语）。

## 七 · 联机方案（极简定案）

**取舍**：不做公共大厅房间列表——本作是莲友私局共修，陌生人房列表带来审核与骚扰负担，与气质不合。
「大厅」＝一屏三事：**开新房 / 房号入房 / 最近的房**；传播靠**邀请链接**，不靠公开曝光。

- **大厅**（题屏「联机同修」进入）：名号（一次填写永久记住）→ 大按钮「开新房」；或房号+入房；下方「最近的房」至多三枚（实时显示 在线/在房 人数、已开局），点即再入。
- **邀请**：房面板「邀请」键→系统分享面板（微信可直转），退化为复制文案+链接；链接形如 `game.foyue.org/#r=AB3D`——莲友点开即弹大厅、房号已填，写名号一键入房，入房即自动入局。
- **轮次**：开局后按座次轮掷；未轮到者掷轮压暗、名单亮当前者；贈掷续掷不交轮（依赠掷本体规则定稿）。
- **断线**：自动重连回原座、棋况保留；离线者名单置灰不除名。
- **聊天**：左下「聊」浮钮+未读角标；名单与聊天同面板，不另开窗。
